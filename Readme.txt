Instukcja do uruchomienia na systemach linux:

	Sklonować projekt : 
	~ git clone git@github.com:Adamszym12/country-search-engine.git
	
	W folderze country-search-engine->server skorzystać z composera
	~ composer install
	
	uruchomić serwer
	~ symfony server:start
	
	Przejść do country-search-engine->client->script.js, w pierwszym wierszu podać adres 	lokalnego serwera
	
	Przejść do  country-search-engine->client->index.html
